## 상태 범례

| 표기 | 의미 |
|------|------|
| **완료** | 명세 대응 화면·핵심 UX가 mockup에 구현됨 (동작은 목 데이터 기준) |
| **부분** | 일부 UI·스텁만 있거나, 명세 대비 범위가 제한됨 |
