## 2. 글로벌 네비게이션 구조

### 사이드바 (Primary Navigation)

```
┌──────────────────────┐
│  Assets Mate 로고      │
├──────────────────────┤
│  📊 대시보드          │
│  🔍 검색 & 탐색       │
│  🛡 브랜드 거버넌스    │
│  🤖 AI Creative      │
│  🤝 협업 & 승인       │
│  📱 소셜 리사이즈      │
│  ⚡ 에셋 최적화       │
├──────────────────────┤
│  ⚙ 설정              │
│  👤 프로필            │
└──────────────────────┘
```

### 상단 헤더 (Global Actions)

```
┌─────────────────────────────────────────────────────┐
│  [브레드크럼]          [글로벌 검색바]    [알림] [프로필] │
└─────────────────────────────────────────────────────┘
```

- **글로벌 검색바**: 어느 페이지에서든 에셋 빠른 검색 가능
- **알림**: 승인 요청, 라이선스 만료, 배포 결과 등 알림 목록
- **프로필**: 사용자 정보, 로그아웃

### 서브 네비게이션 (Secondary Navigation)

각 Theme 영역 진입 시 해당 영역의 하위 메뉴를 탭 또는 서브 사이드바로 표시

예시 - 검색 & 탐색 진입 시:
```
[통합 검색] [비주얼 검색] [색상 검색] [텍소노미] [중복 관리] [다국어 그룹]
```

## 4. 화면 간 네비게이션 플로우

### 핵심 플로우 1: 캠페인 런칭

```
대시보드 → 검색 (에셋 탐색)
  → 에셋 상세 뷰
  → 소셜 리사이즈 (채널별 변환)
  → 브랜드 검사 실행
  → [위반 시] AI 브랜드 수정 (Firefly 자동 보정)
  → 리뷰 & 승인 (승인 요청)
  → 소셜 캘린더 (배포 예약)
  → 배포 이력 (결과 확인)
```

### 핵심 플로우 2: 에이전시 협업

```
대시보드 → 캠페인 워크스페이스
  → 에이전시 포털 (에셋 공유)
  → 리뷰 & 승인 (시안 리뷰)
  → [위반 시] AI 브랜드 수정 제안
  → 버전 비교 (수정 전/후)
  → 승인 상태 대시보드 (현황 파악)
```

### 핵심 플로우 3: 브랜드 관리 + AI 보정

```
대시보드 → 브랜드 대시보드
  → 브랜드 검사 상세 (개별 에셋)
  → [위반 에셋] AI 수정 요청 (Firefly)
  → AI Creative Inbox에 Pending 등록
  → Inbox에서 전/후 비교 확인
  → Approve / Reject / Request Changes
  → 라이선스 관리 (만료 추적)
  → 금지 에셋 관리
```

### 핵심 플로우 4: 다채널 배포

```
대시보드 → 검색 (에셋 선택)
  → 소셜 리사이즈 (원클릭 리사이즈)
  → [비율 차이 큰 채널] Generative Expand (AI 확장)
  → 스마트 리사이즈 프리뷰 (채널별 확인)
  → 소셜 캘린더 (스케줄링)
  → 다채널 동시 배포
  → 배포 이력 (추적)
```

### 핵심 플로우 5: AI 기반 에셋 변형 생성 (신규)

```
에셋 상세 뷰 → AI Creative Studio
  → 자연어 프롬프트 입력 ("겨울 시즌 버전으로")
  → Custom Model 기반 변형 후보 4개 생성
  → 후보 중 선택 → 브랜드 검사 자동 실행
  → [통과 시] 저장 → 승인 프로세스 진행
```

## 6. 반응형 브레이크포인트

React Spectrum S2의 내장 브레이크포인트를 활용한다.

| 브레이크포인트 | 해상도 | 레이아웃 변경 |
|---------------|--------|--------------|
| xs | < 640px | 사이드바 숨김, 바텀 네비, 단일 컬럼 |
| sm | 640-767px | 사이드바 숨김, 바텀 네비, 2컬럼 그리드 |
| md | 768-1023px | 축소 사이드바 (아이콘만), 2-3컬럼 그리드 |
| lg | 1024-1279px | 전체 사이드바, 3-4컬럼 그리드 |
| xl | ≥ 1280px | 전체 사이드바, 4-5컬럼 그리드, 상세 패널 |
